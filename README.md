# 🏦 SmartBankSystem — Banking Management System

> A full-featured console-based banking application built in **Java**

---

## 🧠 What is this?

SmartBankSystem is a Java-based banking management application that simulates core banking operations — account creation, deposits, withdrawals, balance inquiries, and transaction history — using efficient data structures and secure logic.

---

## ✨ Features

- ✅ Create and manage bank accounts
- ✅ Deposit & withdraw funds with validation
- ✅ Real-time balance inquiry
- ✅ Full transaction history per account
- ✅ Secure account authentication
- ✅ Efficient data handling using Java Collections

---

## 🛠️ Tech Stack

| Layer | Technology |
|---|---|
| Language | Java |
| Concepts | OOP, Data Structures, Collections |
| Interface | Command Line (CLI) |

## 🚀 Getting Started

### Prerequisites
- Java JDK 11+
- Any IDE (IntelliJ IDEA / Eclipse / VS Code)

### Run locally

```bash
# Clone the repo
git clone https://github.com/sanskritika2409/SmartBankSystem.git
cd SmartBankSystem

# Compile
javac Main.java

# Run
java Main
```
